# Развертывание проекта на другом ПК

## Требования

- Windows 10/11.
- Node.js 22 LTS или новее.
- npm, который устанавливается вместе с Node.js.

Backend использует встроенный модуль `node:sqlite`, поэтому Node.js 20 для этого проекта недостаточно.

## Самый простой запуск

1. Распакуйте проект в любую папку.
2. Откройте папку проекта.
3. Запустите `start-local.bat` двойным кликом.
4. Дождитесь установки зависимостей при первом запуске.
5. Используйте адрес, который покажет запускатель. Обычно это `http://127.0.0.1:5174/`.

Запускатель сам:

- проверит наличие Node.js;
- установит зависимости через `npm install`, если папки `node_modules` нет;
- выберет свободные порты для backend и frontend;
- создаст `.env.local` с адресом backend;
- запустит backend и frontend в отдельных окнах;
- откроет сайт в браузере.

## Остановка

Закройте окна:

- `Sivka-Burka Backend`;
- `Sivka-Burka Frontend`.

## Запуск из терминала

```powershell
npm run start:local
```

## Ручной запуск

Терминал backend:

```powershell
$env:BACKEND_PORT="8090"
npm run backend:dev
```

Терминал frontend:

```powershell
$env:VITE_API_BASE_URL="http://127.0.0.1:8090/api"
$env:VITE_USE_MOCK_API="false"
npm run dev -- --host 127.0.0.1 --port 5174
```

## Что можно не отправлять в архиве

- `node_modules/`
- `dist/`
- `.env.local`
- `backend/.data/`, если не нужно передавать сохраненные заявки и изменения.

Все эти данные можно восстановить или создать заново при запуске.
